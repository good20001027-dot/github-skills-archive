# GitHub Skills Archive

This repository indexes public GitHub repositories that expose Codex-style `SKILL.md` files.

The archive is metadata-first: it records source repository, file path, discovered name/description snippets, and license status where known. Full skill contents should only be copied when the source license permits redistribution.

## Structure

- `data/skills-index.json` - machine-readable index of discovered skills.
- `docs/methodology.md` - collection rules and safety notes.

## Current Status

Initial seed index created from GitHub code search results. Next updates can expand search queries, verify licenses, and classify skills by domain.
